## Default Rule

If a file, directory, package, module, artifact, or generated distribution is **not expressly designated as Open Material**, it is **Restricted Material**.

The following categories are designated as **Open Materials** unless otherwise marked:
- `/docs/public/**`
- `/examples/**`
- `/sdk/**`
- `/client-libraries/**`
- `/schemas/public/**`
- any file expressly marked **"ZIPPY OPEN MATERIAL"**

## Restricted Materials

The following categories are designated as **Restricted Materials** unless Licensor expressly states otherwise in writing:
- all files not expressly listed as Open Materials
- `/enterprise/**`
- `/internal/**`
- `/private/**`
- `/ops/**`
- `/deployment/**`
- premium modules
- enterprise connectors
- internal policy engines
- proprietary routing heuristics not expressly marked open
- secrets, credentials, keys, environment templates containing confidential values

Restricted Materials remain proprietary to Zippy Technologies LLC and may not be modified or redistributed except as expressly allowed by `LICENSE.md` or a separate written agreement.
